# 🚀 50 Conceptos Básicos para Aprender a Programar

> El curso directo, sin relleno, para pasar de cero a entender cómo piensa un programador.
> Ejemplos en **JavaScript**, el lenguaje que corre en todos los navegadores del mundo.

---

## ¿Para quién es este curso?

- Para quien **nunca ha escrito una línea de código** y quiere empezar bien.
- Para quien ya tocó algo de programación pero tiene **huecos en los fundamentos**.
- Para quien quiere una **referencia rápida y clara** de los 50 conceptos que de verdad importan.

No necesitas instalar nada para empezar: puedes ejecutar todos los ejemplos en la **consola de tu navegador** (abre las herramientas de desarrollador con `F12` y ve a la pestaña *Console*).

---

## Cómo usar este curso

1. Sigue las lecciones **en orden**. Cada una se apoya en la anterior.
2. **Escribe el código tú mismo.** No copies y pegues: teclear fija el aprendizaje.
3. Haz el **ejercicio** del final de cada lección antes de pasar a la siguiente.
4. Si algo no encaja, vuelve atrás. Repetir es normal y necesario.
